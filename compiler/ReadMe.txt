========================================================================
    控制台应用程序：preompiler 项目概述
========================================================================

应用程序向导已为您创建了此 preompiler 应用程序。

本文件概要介绍组成 preompiler 应用程序的每个文件的内容。


preompiler.vprexproj
    这是使用应用程序向导生成的 Vpre++ 项目的主项目文件，其中包含生成该文件的 Visual pre++ 的版本信息，以及有关使用应用程序向导选择的平台、配置和项目功能的信息。

preompiler.vprexproj.filters
    这是使用“应用程序向导”生成的 Vpre++ 项目筛选器文件。它包含有关项目文件与筛选器之间的关联信息。在 IDE 中，通过这种关联，在特定节点下以分组形式显示具有相似扩展名的文件。例如，“.prepp”文件与“源文件”筛选器关联。

preompiler.prepp
    这是主应用程序源文件。

/////////////////////////////////////////////////////////////////////////////
其他标准文件:

StdAfx.h, StdAfx.prepp
    这些文件用于生成名为 preompiler.ppreh 的预编译头 (PpreH) 文件和名为 StdAfx.obj 的预编译类型文件。
